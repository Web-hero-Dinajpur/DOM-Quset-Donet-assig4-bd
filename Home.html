Document Object Model ?

The DOM (Document Object Model) is a programming interface for web documents, essentially acting as a representation of a webpage that can be manipulated with code. When a browser loads an HTML document, it parses the HTML and creates a tree-like structure of nodes, where each part of the document (elements, attributes, and text) is represented as an object or "node" within the DOM.

Key Features of JavaScript

Client-Side Language: JavaScript is most often run in the browser (on the client side) to interact with the webpage after it has been loaded.
Interactivity: It allows developers to respond to user actions (like clicking, typing, or scrolling) and manipulate the DOM in real-time.
Dynamic: JavaScript can modify the content, structure, and styling of a webpage dynamically, without needing to reload the page.
